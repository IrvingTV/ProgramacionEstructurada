I.I Aprendizaje Basado en Problemas: Instalación de Visual Studio Code Contexto:

Desafío

Objetivo:

El objetivo de esta actividad es que los estudiantes sean capaces de instalar y configurar VS Code de manera autónoma, resolviendo los problemas que puedan surgir durante el proceso.

Actividades:

1. Presentación del problema:

Enlace a la página oficial: https://code.visualstudio.com/ 

Identificar requerimientos (Memoria RAM, Almacenamiento, procesador y versión adecuada)

2. Investigación y resolución de problemas:

Los estudiantes investigan en línea (foros, tutoriales, documentación oficial) para resolver los problemas que encuentren durante la instalación y configuración de VS Code.

Se les anima a compartir sus hallazgos y soluciones con sus compañeros.

3. Elaboración del reporte:

Los estudiantes elaboran un reporte escrito que describa el proceso de instalación y configuración de VS Code, incluyendo:

  • Los pasos que siguieron para instalar VS Code.
  • Los problemas que encontraron y cómo los resolvieron.
  • La configuración inicial de VS Code (extensiones instaladas, temas, etc.).
  • Capturas de pantalla que ilustren el proceso y la configuración final.
  • Conclusiones y recomendaciones personales sobre el proceso de instalación.

 4. Entrega y evaluación:

Los estudiantes entregan el reporte escrito a través de la plataforma virtual del curso.
El profesor evalúa los reportes, considerando la claridad de la descripción del proceso, la calidad de las soluciones propuestas y la presentación general del reporte.

Recursos necesarios:

  • Computadoras personales con acceso a internet.
  • Página oficial de descarga de Visual Studio Code.
  • Foros de discusión en línea (Stack Overflow, Reddit, etc.).
  • Tutoriales y documentación oficial de VS Code.
  • Plataforma virtual del curso (para la entrega de reportes).

Evaluación:

  1. Se evaluará la capacidad del estudiante para resolver problemas de manera autónoma.
  2. Se evaluará la capacidad del estudiante para comunicar información técnica de manera clara y concisa.
  3. Se evaluará la capacidad del estudiante para utilizar recursos en línea de manera efectiva.
