Funciones de Entrada-Salida (I/O)

Los tipos de datos en programación se clasifican en primitivos y extendidos. Cada tipo de dato tiene un espacio de memoria y un rango de valores específico.

Entrada (Input):

Lee datos ingresados por el usuario.

nombre = input("Ingresa tu nombre: ")

Salida (Output):

Muestra datos en la pantalla.

print("Hola, " + nombre)

Archivos:

Lectura y escritura de archivos.

with open("archivo.txt", "r") as archivo:
    contenido = archivo.read()

Características de los Datos Primitivos y Extendidos

Los tipos de datos en programación se clasifican en primitivos y extendidos. Cada tipo de dato tiene un espacio de memoria y un rango de valores específico.

Tipos de Datos Primitivos

Enteros (int):

Espacio de memoria: 4 bytes (32 bits).
Rango de valores: -2,147,483,648 a 2,147,483,647.

Flotantes (float):

Espacio de memoria: 4 bytes (32 bits).
Rango de valores: ±3.4e-38 a ±3.4e+38.

Caracteres (char):

Espacio de memoria: 1 byte (8 bits).
Rango de valores: 0 a 255 (caracteres ASCII).

Booleanos (bool):

Espacio de memoria: 1 byte (8 bits).
Rango de valores: true o false.

Tipos de Datos Extendidos

Cadenas (String):

Espacio de memoria: Variable (depende de la longitud de la cadena).
Rango de valores: Secuencia de caracteres.

Arreglos (Array):

Espacio de memoria: Depende del tipo de dato y el tamaño del arreglo.
Rango de valores: Colección de elementos del mismo tipo.

Estructuras (Struct):

Espacio de memoria: Depende de los tipos de datos que contenga.
Rango de valores: Colección de diferentes tipos de datos.

Conversión entre Tipos de Datos

La conversión de tipos es el proceso de cambiar un valor de un tipo de dato a otro. Puede ser implícita (automática) o explícita (manual).

Conversión Implícita:

Ocurre automáticamente cuando se asigna un valor de un tipo a una variable de otro tipo compatible.

x = 5    # int
y = 2.5  # float
z = x + y  # z es float (conversión implícita)

Conversión Explícita:

Se realiza manualmente utilizando funciones de conversión.

x = 5    # int
y = "10" # string
z = x + int(y)  # z es int (conversión explícita)

Palabras Reservadas

Ejemplos comunes

Las palabras reservadas son términos que tienen un significado especial en un lenguaje de programación y no pueden ser utilizados como identificadores. Ejemplos comunes:

  • Python: if, else, for, while, def, class, return, etc.

  • C: int, float, char, if, else, for, while, return, etc.

Identificadores, Variables y Constantes

Identificadores:

Nombres que se asignan a variables, funciones, clases, etc.
Reglas comunes:

Deben comenzar con una letra o _.
No pueden contener espacios ni caracteres especiales.
No pueden ser palabras reservadas.

Variables:

Espacios de memoria que almacenan valores.
Pueden cambiar su valor durante la ejecución del programa.

x = 10  # x es una variable que almacena el valor 10

Constantes:

Valores que no cambian durante la ejecución del programa.
En algunos lenguajes, se definen con palabras clave como const o final.

PI = 3.1416  # PI es una constante
