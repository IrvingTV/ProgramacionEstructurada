Variables

Una variable es donde se almacenan y se recuperan los datos de un programa.

Un espacio en memoria reservado para un valor.

Reglas para nombrarlas:

  • Comienzan siempre por una letra, seguida de otras letras o números o con guión bajo .
  • Solo pueden contener caracteres alfanuméricos (A-Z, a-z, 0-9, _).
    • num_vistas = 50
  • Se distingue entre mayúscula y minúsculas.
  • No se pueden utilizarse palabras reservadas del lenguaje.
  • Los nombres de variables no pueden comenzar con un número.
  • Para asignar un valor a una variable se utiliza el operador = y para borrar una variable se utiliza la instrucción del.

Asignación de variables 

lenguaje = 'Python'
x = 3.14
y = 3 + 2

# Asignación múltiple
a1, a2 = 1, 2

# Intercambio de valores
a, b = b, a

# Incremento (equivale a x = x + 2)
x += 2

# Decremento (equivale a x = x - 1)
x -= 1

# Valor no definido
x = None
del x
